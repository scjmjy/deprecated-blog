## 字体家族（Font Families）

### 4种基本字体家族

- font-family: serif
- font-family: sans-serif
- font-family: monospace
- font-family: cursive

### `@font-face`

```css
@font-face {
 font-family: "SwitzeraADF"; /* descriptor */
 src: url("SwitzeraADF-Regular.otf");
}
h1 {font-family: SwitzeraADF, Helvetica, sans-serif;} /* property */
```
