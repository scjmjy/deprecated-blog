01.函数定义

Ronnie, 15 June 2019

本节讲解JavaScript和TypeScript中函数定义的方法以及区别。

本节目录：

  • 1. JavaScript中的函数定义
    • 1.1. 基本方式
    • 1.2. 变量赋值方式
    • 1.3. 箭头函数方式
  • 2. TypeScript中的函数定义

1. JavaScript中的函数定义

1.1. 基本方式

function foo(name) {
  return 'foo ' + name;
}

1.2. 变量赋值方式

var foo = function (name) {
  return 'foo ' + name;
}

1.3. 箭头函数方式

  • ES6引入了箭头函数

    var foo = (name, age) => {
  return 'foo ' + name + age;
}

var bar = name => 'bar' + name;

2. TypeScript中的函数定义

  • TypeScript中函数的定义方式和JavaScript基本是一样的
  • 不同的是，TypeScript支持为函数的参数和返回值注释类型
function foo(name:string): string {
  return 'foo ' + name;
}
